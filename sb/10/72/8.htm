Canto 10: The Summum BonumChapter 72: The Slaying of the Demon Jarāsandha

Bhaktivedanta VedaBase: Śrīmad Bhāgavatam 10.72.8

rishīnām pitri-devānām

suhridām api nah prabho

sarveshām api bhūtānām

īpsitah kratu-rād ayam

SYNONYMS

rishīnām — for the sages; pitri — departed forefathers; devānām — and demigods; suhridām — for the friends; api — also; nah — our; prabhoh — master; sarveshām — for all; api — as well; bhūtānām — living beings; īpsitah — desirable; kratu — of major Vedic sacrifices; rāt — king; ayam — this.

TRANSLATION

Indeed, My lord, for the great sages, the forefathers and the demigods, for Our well-wishing friends and, indeed, for all living beings, the performance of this king of Vedic sacrifices is desirable.
