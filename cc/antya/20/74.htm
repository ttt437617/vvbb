Antya-līlāChapter 20: The Śikshāshtaka Prayers

Bhaktivedanta VedaBase: Śrī Caitanya Caritāmrita Antya 20.74

tāńra tyakta 'avaśesha' sańkshepe kahila

līlāra bāhulye grantha tathāpi bādila

SYNONYMS

tāńra — his; tyakta — left out; avaśesha — remainders; sańkshepe kahila — I have described very briefly; līlāra bāhulye — because of the numerousness of the pastimes; grantha — this book; tathāpi — still; bādila — has increased.

TRANSLATION

I have only very briefly described the pastimes of Śrī Caitanya Mahāprabhu not described by Vrindāvana dāsa Thākura. Nevertheless, because those transcendental pastimes are so numerous, the size of this book has increased.
