Antya-līlāChapter 1: Śrīla Rūpa Gosvāmī's Second Meeting With the Lord

Bhaktivedanta VedaBase: Śrī Caitanya Caritāmrita Antya 1.127

rāya kahe, — "nāndī-śloka pada dekhi, śuni?"

śrī-rūpa śloka pade prabhu-ājñā māni'

SYNONYMS

rāya kahe — Śrī Rāmānanda Rāya says; nāndī-śloka pada — please recite the introductory verse; dekhi — so that I can see; śuni — so that I can hear; śrī-rūpa śloka pade — Rūpa Gosvāmī recites the verse; prabhu-ājñā māni' — accepting the order of Śrī Caitanya Mahāprabhu.

TRANSLATION

Rāmānanda Rāya said, "Please recite the introductory verse of the Vidagdha-mādhava so that I can hear and examine it." Thus Śrī Rūpa Gosvāmī, being ordered by Śrī Caitanya Mahāprabhu, recited the verse (1.1).
