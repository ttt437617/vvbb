Madhya-līlāChapter 12: The Cleansing of the Gundicā Temple

Bhaktivedanta VedaBase: Śrī Caitanya Caritāmrita Madhya 12.150

ei līlā varniyāchena dāsa vrindāvana

ataeva sańkshepa kari' kariluń varnana

SYNONYMS

ei līlā — this pastime; varniyāchena — has described; dāsa vrindāvana — Vrindāvana dāsa Thākura; ataeva — therefore; sańkshepa — briefly; kari' — doing; kariluń varnana — I have described.

TRANSLATION

This incident has been described in detail by Vrindāvana dāsa Thākura. Therefore I have described it only in brief.

PURPORT

This is a matter of etiquette. If a previous ācārya has already written about something, there is no need to repeat it for personal sense gratification or to outdo the previous ācārya. Unless there is some definite improvement, one should not repeat.
