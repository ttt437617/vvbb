Ādi-līlāChapter 1: The Spiritual Masters

Bhaktivedanta VedaBase: Śrī Caitanya Caritāmrita Ādi 1.93

"pra-śabdena mokshābhisandhir api nirastah" iti

SYNONYMS

pra-śabdena — by the prefix pra; moksha-abhisandhih — the intention of liberation; api — certainly; nirastah — nullified; iti — thus.

TRANSLATION

"The prefix 'pra' [in the verse from Śrīmad-Bhāgavatam] indicates that the desire for liberation is completely rejected."

PURPORT

This is an annotation by Śrīdhara Svāmī, the great commentator on Śrīmad-Bhāgavatam.
